# 🤖 Plateforme du Club Informatique + Chatbot

![Bannière du projet](https://github.com/CGI-Cellule-Projets/.github/blob/main/assets/info-club-hub.jpg?raw=true)

## 💡 Idée du projet
Une plateforme pour le club informatique intégrant un petit chatbot capable de répondre aux questions des nouveaux membres (ex : “Quand aura lieu la prochaine réunion ?”).  
La plateforme inclut également :
- Une page d’accueil (Hero Section, réalisations, cellules du club, nombre de membres, etc.)
- Une section actualités / blog
- Un tableau de bord de gestion interne
